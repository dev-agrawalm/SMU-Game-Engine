2D Platformer Engine

/////////////////////
/How to Use         /
/////////////////////
To run the game, just open the .exe file in the Run folder and make sure it has the fmod dll files.

Main Menu:
Esc: Quit Game

Editor:
Left mouse button on empty tiles: 		Paint tiles/ place entities in the level
Left mouse button on painted tiles/entities: 	Drag/drop entities/tiles
Alt (Hold) + left mouse button:			Duplicate tiles/entities
Shift (Hold) + left mouse button:		Multi select tiles/entities
Select tiles/entities + Delete: 		Delete selection
Ctrl (Hold) + left mouse button:		Erase tiles/entities
W (Hold) + mouse click:				Paint tiles in a box"
E (Hold) + mouse click:				Paint tiles in a line
Ctrl + Z:					Undo
Ctrl + X:					Redo (upto last 10 actions)
Right mouse button: 				Pan the level
Mouse wheel: 					Zoom in and out
Enter:	 					Switch between editing and playing
Esc: 						Go back to main menu

Play mode:
A/D:						Move mario
W/Spacebar:					Jump

F1: Toggle entity debugging (will show cosmetic and physics AABB2 of the entity)
F2: Erase all tiles and entities from the level
F3: Erase all entities from the level
F4: Erase all tile data from the level
F6: Hide Palette window
F7: Hide Inspector window

~: Open dev console

/////////////////////
/Known Issues       /
/////////////////////
The player can get stuck to walls. There is no collision between entities.
